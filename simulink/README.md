# Simulink Model

`ViscoelasticBodies.slx` contains the Simulink artifact associated with the viscoelastic-body modeling part of the repository.

Open from MATLAB:

```matlab
open_system('simulink/ViscoelasticBodies.slx')
```

The scripted MATLAB demos in `src/` are independent from this Simulink model, so the numerical examples can be reviewed without opening Simulink.
